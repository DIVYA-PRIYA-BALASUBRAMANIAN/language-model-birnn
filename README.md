# Language Modeling using Stacked Bidirectional RNNs

## Overview
This project implements a Language Model using stacked Bidirectional LSTM networks.

The model predicts the next word in a sentence based on previous words.

## Tech Stack
Python
TensorFlow
Keras
Numpy

## Model Architecture
Embedding Layer
Bidirectional LSTM Layer
Bidirectional LSTM Layer
Dense Softmax Layer

## How to Run

Install dependencies

pip install -r requirements.txt

Train model

python train.py

Predict

python predict.py
